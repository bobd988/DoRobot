## 坐标系定义

为了避免混淆，请确认您的坐标系定义：

### 机械臂坐标系（从机械臂基座看）

```
        Z (向上)
        ↑
        |
        |
        |________→ X (向前)
       /
      /
     ↙ Y (向左)
```

- **向前**：X 正方向（远离基座）
- **向后**：X 负方向（靠近基座）
- **向左**：Y 正方向
- **向右**：Y 负方向
- **向上**：Z 正方向
- **向下**：Z 负方向

### VR 手柄坐标系（从您的视角）

```
        Y (向上)
        ↑
        |
        |
        |________→ X (向右)
       /
      /
     ↙ Z (向前，远离身体)
```

**请确认这个定义是否正确**：是 / 否
